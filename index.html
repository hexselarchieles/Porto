WEB PROJECT

Event Management

Event Management App adalah aplikasi berbasis website (desktop) yang dibuat sebagai proyek akhir dengan tujuan untuk mempermudah pengguna dalam mengelola dan menyusun berbagai kegiatan atau acara. Melalui tampilan yang responsif dan mudah digunakan, aplikasi ini menyediakan fitur untuk menambahkan, mengedit, dan menghapus data event, mengatur jadwal, serta mencatat detail penting seperti lokasi dan peserta. Aplikasi ini dibangun menggunakan PHP dan terintegrasi dengan database SQL, sehingga seluruh data event dapat tersimpan dengan aman, terstruktur, dan dapat diakses dengan mudah melalui perangkat desktop.
